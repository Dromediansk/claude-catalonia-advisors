# Buses

## TMB city buses

- Run by **TMB**, fully integrated with metro/FGC/tram passes (T-Casual, T-Usual, T-Familiar, Hola Barcelona, T-jove all valid).
- **Pay**: validate card at the front; or contactless single fare ~€2.90 `[verify]`.
- **Hours**: roughly 05:00–23:00 (varies by line) `[verify]`. After that → Nitbus.

### Useful tourist routes

| Line | Goes to |
|---|---|
| **24** | Pl. Catalunya → Park Güell (main route up) |
| **116** | Vallcarca → Park Güell (small neighbourhood bus to the back gate) |
| **150** | Pl. Espanya → Montjuïc Castle (via the hill) |
| **V13, V15, V17** | Diagonal vertical routes — handy for crossing the city |
| **H6, H8, H10, H12, H16** | Horizontal high-frequency network |
| **D20** | Pg. Marítim → beach + Forum |
| **39, 42** | Hospital de Sant Pau (Modernisme) |

The **H** and **V** lines are the New Bus Network (Nova Xarxa) — high-frequency, grid-based. Easiest for tourists to navigate.

## Nitbus (night buses)

- **18 routes** numbered **N0–N17** `[verify]`.
- Run roughly **22:00–06:00** (varies) `[verify]`.
- **All lines except N0 stop at Plaça de Catalunya** (transfers hub). N0 is a city-centre loop.
- Frequency ~20 min `[verify]`.
- Covered by TMB passes.

### Useful Nitbus routes for tourists

| Line | Route |
|---|---|
| **N17** | Pl. Catalunya ↔ Aeroport T1 (~60 min) `[verify]` |
| **N18** | Pl. Catalunya ↔ Aeroport T2 (~60 min) `[verify]` |
| **N9, N15** | Beach / Barceloneta routes |
| **N0** | Central loop, transfer hub |

## Intercity coaches (long-distance buses)

Two main terminals:

### Estació d'Autobusos Barcelona Nord (Estació del Nord)

- Address: C/ d'Alí Bei, 80
- Metro: **Arc de Triomf (L1)**
- Main companies:
  - **ALSA** — Spain-wide & international (Madrid, Zaragoza, Valencia, Granada, Andorra, Toulouse)
  - **Moventis Sarfa** — Costa Brava: **Cadaqués**, L'Escala, L'Estartit, Lloret de Mar, Palafrugell, Palamós, Platja d'Aro, Roses
  - **Sagalés** — Catalan interior, Vic, Olot
  - **FlixBus** — discount intercity & international
  - **Avanza**
- Costa Brava tickets: buy at Moventis Sarfa desk (Hall C, offices 2–3) `[verify]` or online at moventis.es

### Estació de Sants (bus terminal next to the train station)

- ALSA, Eurolines, FlixBus international departures.
- Metro: **Sants Estació (L3, L5)**.

### Useful operators by region

| Operator | Strength |
|---|---|
| **ALSA** | National network — Madrid, Valencia, Andorra, Cadaqués via Figueres |
| **Moventis Sarfa** | Costa Brava direct |
| **MonBus** | Costa Brava, Pyrenees, Lleida province |
| **Sagalés** | Catalan interior |
| **Direct Bus / Andbus** | Andorra |
| **FlixBus** | Cheap intercity & cross-border |
| **Eurolines** | International long-haul |
